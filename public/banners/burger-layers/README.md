# Burger layer images

Add your PNG layers here (transparent background). Order in the burger assembly:

1. `bun-bottom.png` - Bottom bun
2. `patty.png` - Burger patty
3. `cheese.png` - Cheese slice
4. `mushrooms.png` - Mushrooms
5. `sauces.png` - Sauces
6. `bun-top.png` - Top bun

Edit `lib/burger-layers.ts` to change paths or add more layers.
